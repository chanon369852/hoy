# RGA Dashboard Frontend (React + Tailwind CSS)

## Installation

```bash
cd frontend-react
npm install
```

## Development

```bash
npm run dev
```

This will start Vite dev server on http://localhost:3000

## Build for Production

```bash
npm run build
```

This will create a `frontend-build` folder that can be served by the backend.

## Features

- ✅ React 18 with React Router
- ✅ Tailwind CSS for styling
- ✅ Axios for API calls
- ✅ Authentication context
- ✅ Protected routes
- ✅ Responsive design
- ✅ All pages: Login, Register, Dashboard, Clients, Users, Products, Orders, Alerts, Ad Metrics, SEO Metrics, Uploads
